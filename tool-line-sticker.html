使用流程：
1. 上傳整張大圖
2. 預設用「網格模式」（4×4 均分）
  • 拖藍色分隔線可整列/整欄一起動
  • 右側可改欄數、列數
  • 🔒 解鎖某格 → 完全獨立，不受網格線影響
3. 不規則排版 → 切到「自動偵測」模式
4. 框錯了？
  • 點框 = 排除（變紅）
  • 拖框到另一框上 = 合併
  • 雙擊框 = 編輯（拉大小、× 刪除）
5. 切換「輸出角色」→ 加入所選
6. 主體加描邊 → 顏色淺的貼圖更顯眼
